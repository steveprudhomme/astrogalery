# GNU Astro Galery
**Version : 00.00.07**

**GNU Astro Galery** est un générateur de galerie Web statique (HTML5 / CSS3 / Bootstrap 5) destiné aux images produites par le **télescope intelligent Seestar S50**.

Le projet vise à créer une galerie :
- esthétique,
- scientifiquement documentée,
- pérenne,
- facilement publiable,

en mettant la **page objet** au centre de la navigation.

## Fonctionnalités

- Génération d’un **site Web statique complet**
- Page d’accueil :
  - toutes les images,
  - triées du **plus récent au plus ancien**,
  - recherche textuelle instantanée,
  - filtres dynamiques par type et catalogue
- **Clic sur l’image → page objet**
- Pages objet individuelles comprenant :
  - image principale prédominante,
  - auteur et licence,
  - métadonnées détaillées de l’image,
  - caractéristiques astronomiques,
  - image astrométrique cliquable,
  - **carte stellaire de type atlas**
- Enrichissement automatique depuis :
  - **SIMBAD**
  - **catalogue Messier (XLSX local)**
  - **catalogue d’objets divers (XLSX multi-feuilles)**
- Astrométrie optionnelle via **astrometry.net**
- Système de **cache complet** (Simbad, astrométrie, cartes)
- SEO intégré :
  - sitemap.xml
  - robots.txt
  - OpenGraph
  - JSON-LD (schema.org / ImageObject)

## Philosophie générale

GNU Astro Galery est conçu autour de principes simples :

- **Une image = un objet astronomique**
- **Une page objet = une fiche documentaire complète**
- **Aucune dépendance serveur**
- **Traçabilité maximale des données**
- **Respect des standards ouverts**
